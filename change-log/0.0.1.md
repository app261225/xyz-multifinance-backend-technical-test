# Changelog Tahap 1: Security & Connection Establishment
Dalam tahap ini, kamu telah melakukan transformasi dari koneksi database standar ke koneksi tingkat enterprise:
1. Environment Setup: Mengisolasi kredensial (host, user, password) ke dalam .env agar tidak hardcoded (Keamanan dasar).
2. Certificate Handshake: Memuat sertifikat otoritas (ca.pem) ke dalam aplikasi untuk memverifikasi bahwa server database Aiven benar-benar aman dan bukan serangan Man-in-the-Middle.
3. Driver Registration: Mendaftarkan konfigurasi TLS khusus ke dalam driver go-sql-driver/mysql agar Go mengenali protokol SSL yang diminta Aiven.
4. DSN Integration: Menyusun Data Source Name dengan parameter tls=custom-tls dan port khusus (16328) untuk membuka jalur komunikasi terenkripsi.
5. ORM Initialization: Menghubungkan GORM dengan koneksi fisik tersebut untuk mempermudah manipulasi data di tahap selanjutnya.
